## ✍ 진행방식

- 매주 평일 스터디 진행
  - 시간은 오전 10시
- 일마다 정해진 분량의 강의를 듣고 정리 후 토의 진행
  - 강의를 들으며 정리해볼 내용
    1. 진행은 됐으나 이해가 되지 않은 것
    2. 이해는 됐으나 잘 진행되지 않은 것
    3. 이해와 진행 모두 되지 않은 것
  - 공부 후 정리한 파일은  `.md` 를 업로드 한다.
- 자신의 이름으로 branch를 생성해 PR을 날리고 다른 사람이 확인한 뒤 merging한다.

> ➕ 추가적으로 더 공부하고 진행하여 알게된 것이 있다면 공유하기

<br/>

### commit 컨벤션

```
  docs: md파일 작성
  edit: 내용 수정
  del: 파일 혹은 폴더 삭제
  rename: 파일명 수정
```
